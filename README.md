## Deskripsi Tugas
Buatlah program Java yang terkoneksi dengan database MySQL. Berikut
spesifikasi program yang harus dibuat:
â— Program bebas, kecuali program Mahasiswa dan Book Author

â— Terdapat proses Create, Read, Update, dan Delete data

â— Terdapat proses Autentikasi (Login, Register) *

â— Menggunakan minimal 2 tabel pada database

â— Harus terdapat minimal 1 properti gambar pada class yang dibuat (gambar
akan ditampilkan pada UI)

â— Terdapat pergantian screen pada UI

â— Terdapat button navigasi untuk beralih screen

â— List data ditampilkan menggunakan card (JPanel)
contoh card dengan JPanel link

* Boleh login dan/atau register. Tidak perlu ada enkripsi password
